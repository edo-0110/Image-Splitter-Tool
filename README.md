# Image-Splitter-Tool
這是一個簡單、直覺的 Web 應用程式，專門用於將單張圖片從中間切割成左右兩半。非常適合需要快速處理圖片並下載結果的使用者。(功能逐漸增加中)
# 📸 圖片自動切割工具 (Image Splitter Tool)

[![Python Version](https://img.shields.io/badge/python-3.7%2B-blue)](https://www.python.org/)
[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://streamlit.io/)


## ✨ 功能特性 (Features)

*   **一鍵切割**：自動將圖片從垂直中心線切開。
*   **即時預覽**：上傳後立即在網頁上看到切割後的結果。

*   **無需安裝**：透過 Streamlit Cloud 部署，直接透過瀏覽器使用。
*   **快速下載**：切割完成後可直接下載左半部與右半部的圖片。

## 🛠️ 使用方法 (How to Use)

1.  **開啟網頁**：點擊 [here](https://image-splitter-tool-e2rhg5ecfedeykywwrwrne.streamlit.app/)進入應用程式。
2.  **上傳圖片**：點擊上傳按鈕，選擇你想處理的圖片檔案（支援 JPG, PNG 等）。
3.  **查看結果**：系統會自動處理，你可以在畫面上看到左、右兩部分的預覽。
4.  **下載圖片**：點擊下方的下載按鈕，將切割後的圖片儲存到電腦中。
